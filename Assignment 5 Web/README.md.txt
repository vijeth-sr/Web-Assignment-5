SoccerSplash is a simple website with multiple sections. It includes:

Homepage (Page 1): The landing page of the website features a navigation bar with links to different sections, a banner with a search box, and some introductory text.

Media Section (Page 2): This section features audio and video elements, as well as a "About Us?" dropdown details.

Customers Section: A section showcasing happy customers' testimonials.

Teamwear Section (Page 3): Displays teamwear products with images and prices.

Footer: The footer includes contact information, social media links, and a link to an external website.

Project is built using html, scss features and javascript
